# PairHub

PairHub is a place to find remote pair programming partners, either peers or paid experts.

## Tech stack
React, Next.js, Prisma, Postgres, Tailwind

### Development

Run database with
```
docker-compose up
```

Start dev process with 

```
npm run dev
```
